# Work Order Schedule Timeline

Angular standalone timeline UI for planning work orders across manufacturing work centers.

## Run Locally

```bash
npm install
ng serve
```

Open `http://localhost:4200`.

Stress mode (50 work centers / 10,000 orders):

```bash
http://localhost:4200/?stress
```

## Libraries Used

- `@ng-select/ng-select`: status picker in the create/edit panel with a compact selectable dropdown UX.
- `@ng-bootstrap/ng-bootstrap`: start/end date pickers (`ngb-datepicker`) for fast date selection without custom calendar logic.

## Architecture Notes

- Single feature component: `TimelineComponent` contains timeline rendering, interaction handlers, and form logic.
- Time columns are generated by timescale (`day | week | month`) into `visibleColumns` with `startUtcDay/endUtcDay` bounds.
- Work-order bars are positioned from date math (`utcDayToPx`) and sized by start/end span.
- Click-to-create uses inverse math (`pxToUtcDay`) to convert row click position into `YYYY-MM-DD`.
- Create/Edit panel uses a reactive form storing ISO dates (`startsAtIso`, `endsAtIso`) with datepicker conversion helpers.
- Overlap validation checks only work orders in the same work center and skips the currently edited order.
